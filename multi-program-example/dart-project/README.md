# Dart CLI Example: Codama-Dart Renderer Showcase

This project is a sample command-line application written in Dart. It demonstrates how the **codama-dart renderer** works in the terminal, providing a practical example for integrating and showcasing its features.

## Features

- Basic argument parsing
- Terminal-based rendering using codama-dart
- Example CLI entrypoint in [`bin/`](bin/)
- Easily extensible for your own use cases

## Getting Started

1. **Install dependencies:**
```sh
dart pub get
```

2. **Run the CLI**
```sh
dart run
```

## Project Structure
- `bin/`: CLI entrypoint and main logic
- `clients/`: Example client implementations
- `utils/`: Utility functions and helpers
- `keypairs/`: Keypair files for testing
